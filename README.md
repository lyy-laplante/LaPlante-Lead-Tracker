# LaPlante Lead Tracker

A Next.js lead tracker deployed on Netlify and backed by Netlify Database.

## Access

This version has no login or PIN screen. Opening the deployed site goes directly to the lead tracker.

## Deploy

1. Upload all files and folders from this project to the root of the GitHub repository.
2. Commit the changes.
3. Netlify should deploy automatically.
4. Confirm the latest deployment is Published.

## Important

Anyone who has the live site URL can access and change lead information. Do not store sensitive personal information unless access protection is added later.
